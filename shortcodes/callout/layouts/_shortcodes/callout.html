{{/* Renders a paired admonition / callout block: {{< callout "warning" "Heads up" >}} Body **markdown** here -- nested shortcodes and render hooks all work. {{< /callout >}} This is a UNIVERSAL, UNSTYLED admonition: it emits semantic BEM markup, data attributes, and CSS-custom-property tone NAMES, and ships ZERO CSS. The consuming site owns all visual styling. Fifteen first-class types plus true aliases and arbitrary custom-type passthrough, opt-in ARIA, native collapsibility, verbatim user titles, and a block-level body (never a bare

wrapper). Parameters: - type (positional 0, or type=): callout type. Default "note". May be a first-class type, an alias, or ANY custom slug (passed through literally -- designers define unlimited custom callout styles). See resolve-type.html and the README Types table. - title (positional 1, or title=): head title. * Omitted -> the canonical type label (i18n-able), passed through `title` casing, e.g. "warning" -> "Warning". * Supplied -> emitted VERBATIM (never re-cased), so "iOS quirks" stays "iOS quirks". A non-string value is stringified (title=123 renders "123"). * title="" (or a whitespace-only title) -> suppresses the head entirely (icon + title row omitted). - collapsible (bool): render as a native

/ disclosure. Truthy tokens are true/1/yes/on in any casing; falsy tokens are false/0/no/off or omission. An unrecognized token is treated as false with a one-shot warning. - open (bool): when collapsible, start expanded (adds the `open` attribute). Same token rules as collapsible. - role (string): opt-in ARIA override. One of "note", "alert", "status", "none". Default is the passive role from resolve-type.html ("note"). Use "alert"/"status" ONLY for genuinely dynamic, client-updated notices -- they are live regions and would make assistive tech announce static prose. "none" emits no role attribute. - icon: icon override. A first-class icon NAME (rendered via icon.html, matched case-insensitively), a single emoji (emitted as text), or an image reference rendered as . A value counts as an image reference only when it contains "/", starts with http:// or https://, or ends in a known image extension (svg, png, jpg, jpeg, jpe, jif, jfif, gif, webp, avif, ico, bmp, tif, tiff, heic, heif -- the extensions Hugo's image media types cover, plus ico); any other value -- including a dotted name like "brand.v2" -- is a glyph name. Local references resolve as a page resource first, then a global resource; remote URLs are fetched at build time and republished under the site's own origin. icon=false (or any of the tokens "false", "none", "no", "off", "0", "", any casing) suppresses the icon. When omitted, first-class types use their default glyph and custom/unknown types render no icon. - id (string): id attribute on the root element (for anchoring/linking). - class (string): extra class(es) appended to the root element. Usage (a single call uses EITHER all-positional OR all-named arguments -- Hugo forbids mixing the two conventions in one call): callout "tip" callout "danger" "Do not do this" callout type="success" title="" icon="🎉" callout type="my-brand-type" title="Brand note" icon="/images/brand.svg" callout type="warning" collapsible="true" open="true" role="alert" Never errorf: an unknown type is valid (passthrough), an empty body is valid, and a bad role/icon degrades gracefully. warnf is used sparingly. */}} {{- $rawType := .Get 0 -}} {{- if not $rawType -}}{{ $rawType = .Get "type" }}{{- end -}} {{- $type := or $rawType "note" -}} {{/* Title resolution distinguishes three states: a user-supplied value (positional index 1 OR title=, including an explicit empty string => suppress head) versus omitted (=> default label). A missing NAMED argument does not reliably come back as nil from `.Get` when other named arguments are present, so presence is detected with the dedicated argument API: `.IsNamedParams` plus `isset .Params "title"` for the named form (Hugo lowercases named keys), and the positional argument count for the positional form. */}} {{- $titlePresent := false -}} {{- $title := "" -}} {{- if .IsNamedParams -}} {{- if isset .Params "title" -}} {{- $titlePresent = true -}} {{- $title = printf "%v" (.Get "title") -}} {{- end -}} {{- else if ge (len .Params) 2 -}} {{- $titlePresent = true -}} {{- $title = printf "%v" (.Get 1) -}} {{- end -}} {{/* Boolean parameters accept true/1/yes/on in any casing (string or bare bool -- printf "%v" folds both into one comparable string form), with surrounding whitespace trimmed, and false/0/no/off or omission for false. An unrecognized token coerces to false with a one-shot deduplicated warning that echoes the author's original value, matching the module's warn-on-bad-input convention for type, role, and icon. */}} {{- $truthyTokens := slice "true" "1" "yes" "on" -}} {{- $falsyTokens := slice "false" "0" "no" "off" "" -}} {{- $collapsibleInput := printf "%v" (.Get "collapsible" | default false) -}} {{- $collapsibleTok := lower (trim $collapsibleInput " \t\n\r") -}} {{- $collapsible := in $truthyTokens $collapsibleTok -}} {{- if and (not $collapsible) (not (in $falsyTokens $collapsibleTok)) -}} {{- $warnKey := printf "callout:warned-bool:collapsible:%s" $collapsibleTok -}} {{- if not (hugo.Store.Get $warnKey) -}} {{- hugo.Store.Set $warnKey true -}} {{- warnf "[callout] Ignoring unrecognized collapsible=%q (use true/1/yes/on or false/0/no/off, any casing). Treating it as false. See %s" $collapsibleInput .Position -}} {{- end -}} {{- end -}} {{- $openInput := printf "%v" (.Get "open" | default false) -}} {{- $openTok := lower (trim $openInput " \t\n\r") -}} {{- $open := in $truthyTokens $openTok -}} {{- if and (not $open) (not (in $falsyTokens $openTok)) -}} {{- $warnKey := printf "callout:warned-bool:open:%s" $openTok -}} {{- if not (hugo.Store.Get $warnKey) -}} {{- hugo.Store.Set $warnKey true -}} {{- warnf "[callout] Ignoring unrecognized open=%q (use true/1/yes/on or false/0/no/off, any casing). Treating it as false. See %s" $openInput .Position -}} {{- end -}} {{- end -}} {{- $roleParam := lower (.Get "role" | default "") -}} {{/* Icon is a named-only parameter. Detect its presence with isset on the named params map (a missing named arg is not reliably nil from `.Get`), so an omitted icon falls back to the type default while icon="" / icon=false are still honored. */}} {{- $iconPresent := and .IsNamedParams (isset .Params "icon") -}} {{- $iconParam := "" -}} {{- if $iconPresent -}}{{- $iconParam = .Get "icon" -}}{{- end -}} {{- $id := .Get "id" | default "" -}} {{- $class := .Get "class" | default "" -}} {{/* Resolve the type to its canonical form + defaults (alias mapping, default icon, default label, passive role, known/unknown). Shared with the blockquote render hook so [!NOTE] syntax renders identically. */}} {{- $r := partial "callout/resolve-type.html" $type -}} {{- $canonical := $r.canonical -}} {{/* Default-deduplicated warning for unknown types, so a typo surfaces once per build without breaking it. hugo.Store is build-scoped: the first call for a given unknown type warns, the rest stay silent. Known types and aliases never warn. */}} {{- if not $r.known -}} {{- $warnKey := printf "callout:warned-unknown:%s" $canonical -}} {{- if not (hugo.Store.Get $warnKey) -}} {{- hugo.Store.Set $warnKey true -}} {{- warnf "[callout] Unknown callout type %q is rendered as a custom type (class callout--%s, data-callout-type=%q, no default icon). If this was a typo, use a first-class type or an alias; if intentional, define styles for this type in your site CSS. See %s" $canonical $canonical $canonical .Position -}} {{- end -}} {{- end -}} {{/* Head title: a verbatim user value, or the resolver's display-ready default label (already Title-cased or localized; do NOT re-case it). The empty-string user value falls through to head suppression below. */}} {{- $headTitle := "" -}} {{- if $titlePresent -}} {{- $headTitle = $title -}} {{- else -}} {{- $headTitle = $r.label -}} {{- end -}} {{- $showHead := or (not $titlePresent) (ne (trim $title " \t\n\r") "") -}} {{/* A collapsible callout MUST expose a , and a needs a real, author-controlled accessible name (otherwise the browser falls back to a non-localizable "Details" label). So in the collapsible form the head is always rendered, and a suppressed title (title="") falls back to the type's label for the summary. A consumer that wants the title visually hidden does so with CSS, not by dropping the summary. */}} {{- if $collapsible -}} {{- $showHead = true -}} {{- if eq (trim $headTitle " \t\n\r") "" -}} {{- $headTitle = $r.label -}} {{- end -}} {{- end -}} {{/* ARIA role resolution. Default is passive. The role= param can override to a live region (alert/status) for dynamic notices, or "none" to omit. An unrecognized role value is ignored (falls back to the passive default) with a one-shot warning. */}} {{- $role := $r.role -}} {{- $emitRole := true -}} {{- if $roleParam -}} {{- if eq $roleParam "none" -}} {{- $emitRole = false -}} {{- else if in (slice "note" "alert" "status") $roleParam -}} {{- $role = $roleParam -}} {{- else -}} {{- $warnKey := printf "callout:warned-role:%s" $roleParam -}} {{- if not (hugo.Store.Get $warnKey) -}} {{- hugo.Store.Set $warnKey true -}} {{- warnf "[callout] Ignoring unknown role %q (use note, alert, status, or none). Falling back to the passive default. See %s" $roleParam .Position -}} {{- end -}} {{- end -}} {{- end -}} {{/* Icon strategy: - icon=false / "false" / "none" / "no" / "off" / "0" / "" -> suppress. - icon is a single emoji (one grapheme, non-ASCII) -> emit as text. - icon is an image reference -> resolve to an . A value counts as an image reference ONLY when it contains "/", starts with http:// or https://, or ends in a known image extension (svg, png, jpg, jpeg, jpe, jif, jfif, gif, webp, avif, ico, bmp, tif, tiff, heic, heif) -- a bare dotted name such as "brand.v2" is a glyph name, not a path. Remote URLs are fetched at build time and republished under the site's own origin; local values resolve via page resources then global resources. A failed fetch or lookup degrades to no icon with a one-shot warning -- the build never breaks. - icon is a bare name -> render via icon.html (matched case-insensitively; works for first-class names and lets a custom type borrow a built-in glyph). - icon omitted -> default glyph for first-class types, nothing otherwise. */}} {{- $iconMode := "default" -}} {{- $iconName := $r.iconName -}} {{- $iconEmoji := "" -}} {{- $iconResource := "" -}} {{- if $iconPresent -}} {{- $iconStr := trim (printf "%v" $iconParam) " \t\n\r" -}} {{- $iconLower := lower $iconStr -}} {{- $iconIsRemote := or (hasPrefix $iconLower "http://") (hasPrefix $iconLower "https://") -}} {{- $iconHasImageExt := gt (len (findRE `\.(svg|png|jpe?g|jpe|jif|jfif|gif|webp|avif|ico|bmp|tiff?|heic|heif)$` $iconLower 1)) 0 -}} {{- if or (in (slice "false" "none" "no" "off" "0" "") $iconLower) (eq $iconParam false) -}} {{- $iconMode = "none" -}} {{- else if or $iconIsRemote (in $iconStr "/") $iconHasImageExt -}} {{- $iconMode = "resource" -}} {{- $res := "" -}} {{- if $iconIsRemote -}} {{- $remote := try (resources.GetRemote $iconStr (dict "timeout" "30s")) -}} {{- with $remote.Err -}} {{- $warnKey := printf "callout:warned-icon:%s" $iconStr -}} {{- if not (hugo.Store.Get $warnKey) -}} {{- hugo.Store.Set $warnKey true -}} {{- warnf "[callout] Remote icon %q could not be fetched (%s). Rendering without an icon. See %s" $iconStr . $.Position -}} {{- end -}} {{- else -}} {{- with $remote.Value -}}{{ $res = . }}{{ end -}} {{- end -}} {{- else -}} {{- with .Page.Resources.Get $iconStr -}}{{ $res = . }}{{ end -}} {{- if not $res -}} {{- with resources.Get $iconStr -}}{{ $res = . }}{{ end -}} {{- end -}} {{- end -}} {{- if $res -}} {{- $iconResource = $res.RelPermalink -}} {{- else -}} {{- $iconMode = "none" -}} {{- $warnKey := printf "callout:warned-icon:%s" $iconStr -}} {{- if not (hugo.Store.Get $warnKey) -}} {{- hugo.Store.Set $warnKey true -}} {{- warnf "[callout] Icon %q could not be resolved (page resources, assets/, or remote fetch). Rendering without an icon. See %s" $iconStr .Position -}} {{- end -}} {{- end -}} {{- else if and $iconStr (not (findRE `^[\x00-\x7F]+$` $iconStr)) -}} {{/* Contains a non-ASCII rune and is not path-like -> treat the literal as an emoji / symbol (handles multi-codepoint emoji such as ZWJ sequences, which a length test would misclassify). */}} {{- $iconMode = "emoji" -}} {{- $iconEmoji = $iconStr -}} {{- else -}} {{/* A bare ASCII name -> render the named built-in glyph, matched case-insensitively like every other token surface. The glyph set lives in icon.html alone, so membership is probed by rendering: an unknown name yields empty output and degrades to no icon with a one-shot warning, keeping an explicit typo visible. */}} {{- $iconMode = "name" -}} {{- $iconName = $iconLower -}} {{- if eq (trim (partial "callout/icon.html" (dict "name" $iconLower)) " \t\n\r") "" -}} {{- $iconMode = "none" -}} {{- $warnKey := printf "callout:warned-icon:%s" $iconLower -}} {{- if not (hugo.Store.Get $warnKey) -}} {{- hugo.Store.Set $warnKey true -}} {{- warnf "[callout] Unknown icon name %q (not a built-in glyph). Rendering without an icon. See %s" $iconStr .Position -}} {{- end -}} {{- end -}} {{- end -}} {{- end -}} {{- $showIcon := and (ne $iconMode "none") (or (ne $iconName "") (eq $iconMode "emoji") (eq $iconMode "resource")) -}} {{/* Render the inner body as block-level markdown. RenderString takes the markup string LAST; display=block keeps full paragraph structure (so the body is never collapsed into a bare inline

). The full content pipeline runs, so nested shortcodes, render hooks, and markdown all work. An empty body is allowed and simply yields an empty body element. */}} {{- $inner := .Inner | strings.TrimSpace -}} {{- $bodyHTML := "" -}} {{- with $inner -}} {{- $bodyHTML = $.Page.RenderString (dict "display" "block") . -}} {{- end -}} {{/* Assemble the shared rendering context and dispatch to the markup partial, which both this shortcode and the render hook reuse for identical output. */}} {{- $ctx := dict "canonical" $canonical "extraClass" $class "id" $id "emitRole" $emitRole "role" $role "collapsible" $collapsible "open" $open "showHead" $showHead "headTitle" $headTitle "showIcon" $showIcon "iconMode" $iconMode "iconName" $iconName "iconEmoji" $iconEmoji "iconResource" $iconResource "bodyHTML" $bodyHTML -}} {{- partial "callout/render.html" $ctx -}}
