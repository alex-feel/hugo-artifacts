{{/* Renders an inline SVG icon by name for the callout module. Context expectations: - .name (string) - Icon identifier. One of the first-class callout type icon names produced by resolve-type.html: note, info, tip, success, question, important, warning, caution, danger, failure, bug, example, quote, abstract, todo. An unrecognized name renders nothing (no fallback glyph), which is the correct behavior for custom/unknown callout types. - .class (string, optional) - Additional CSS class(es). Contract (identical to the github-repo / hf-space sibling modules so the three read as a family): every icon uses width="1em" height="1em" so it scales with font size, aria-hidden="true" because the type is already conveyed by the visible title and the data-callout-type attribute, and a 24x24 viewBox. Stroke-based glyphs use stroke="currentColor" with no fill; fill-based glyphs use fill="currentColor". The consuming site colors and sizes them through the BEM class hooks and CSS custom properties -- the module ships NO color and depends on NO external icon font. The geometry is original line-art in the Lucide / Tabler / Octicons visual idiom (24x24 grid, 2px stroke, round caps/joins). No third-party SVG assets are bundled. */}} {{- $name := .name -}} {{- $class := .class | default "" -}} {{- $stroke := `width="1em" height="1em" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round" aria-hidden="true" focusable="false"` -}} {{- $fill := `width="1em" height="1em" viewBox="0 0 24 24" fill="currentColor" aria-hidden="true" focusable="false"` -}} {{- if eq $name "note" -}} {{- else if eq $name "info" -}} {{- else if eq $name "tip" -}} {{- else if eq $name "success" -}} {{- else if eq $name "question" -}} {{- else if eq $name "important" -}} {{- else if eq $name "warning" -}} {{- else if eq $name "caution" -}} {{- else if eq $name "danger" -}} {{- else if eq $name "failure" -}} {{- else if eq $name "bug" -}} {{- else if eq $name "example" -}} {{- else if eq $name "quote" -}} {{- else if eq $name "abstract" -}} {{- else if eq $name "todo" -}} {{- end -}}
