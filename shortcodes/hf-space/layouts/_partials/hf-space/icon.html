{{/* Renders an inline SVG icon by name. Context expectations: - .name (string) - Icon identifier: "external", "heart", "chip", "clock" - .class (string, optional) - Additional CSS class(es) All icons use fill="currentColor", aria-hidden="true", viewBox="0 0 16 16", and width/height of 1em for text-relative sizing -- the same contract as the github-repo sibling module, so the two read as a family. The module ships its own SVGs and does NOT depend on an external icon font; the consuming site colors and sizes them through the BEM class hooks. The Hugging Face brand mark itself is emitted as the 🤗 emoji in the variant partials (it is content/branding, not a functional icon), so it is not part of this icon set. */}} {{- $name := .name -}} {{- $class := .class | default "" -}} {{- if eq $name "external" -}} {{- else if eq $name "heart" -}} {{- else if eq $name "chip" -}} {{- else if eq $name "clock" -}} {{- end -}}
