{{/* Formats a number in compact notation (e.g., 1500 -> "1.5k", 1200000 -> "1.2M"). Used for the Space like count, which the Hugging Face Hub reports as a plain integer. Identical formatting contract to the github-repo sibling module so the two read as a family. Context: a numeric value (int, float, or numeric string) Returns a formatted string. A value Hugo's float cast accepts (numbers, numeric strings, and cast coercions such as nil -> 0 and booleans -> 0/1) formats numerically when it is finite and within the plausible count range (|n| < 1e15); any other value -- unparseable, NaN, infinite, or extreme -- is returned stringified as-is. The input is remote-derived, so bad data degrades to verbatim passthrough instead of failing the build or rendering garbage, and the single up-front guarded cast covers every downstream tier comparison. */}} {{- $result := printf "%v" . -}} {{- $n := 0.0 -}} {{- $ok := false -}} {{- with try (float .) -}} {{- if not .Err -}} {{- $n = .Value -}} {{- $ok = true -}} {{- end -}} {{- end -}} {{/* The float cast also accepts NaN, Inf, and exponent-notation extremes; int-casting NaN saturates to MinInt64 and a 1e300 value renders a 300-digit string, so anything non-finite (NaN is the one value unequal to itself) or outside the plausible count range falls back to verbatim passthrough. */}} {{- if and $ok (or (ne $n $n) (ge (math.Abs $n) 1e15)) -}} {{- $ok = false -}} {{- end -}} {{- if $ok -}} {{- $result = printf "%d" (int $n) -}} {{- end -}} {{- if and $ok (ge $n 1000000.0) -}} {{- $d := math.Div $n 1000000.0 -}} {{- $whole := math.Floor $d -}} {{- $frac := math.Round (math.Mul (math.Sub $d $whole) 10) -}} {{- if eq $frac 10.0 -}} {{- $whole = math.Add $whole 1 -}} {{- $frac = 0.0 -}} {{- end -}} {{- if eq $frac 0.0 -}} {{- $result = printf "%.0fM" $whole -}} {{- else -}} {{- $result = printf "%.0f.%.0fM" $whole $frac -}} {{- end -}} {{- else if and $ok (ge $n 1000.0) -}} {{- $d := math.Div $n 1000.0 -}} {{- $whole := math.Floor $d -}} {{- $frac := math.Round (math.Mul (math.Sub $d $whole) 10) -}} {{- if eq $frac 10.0 -}} {{- $whole = math.Add $whole 1 -}} {{- $frac = 0.0 -}} {{- end -}} {{/* A carry that reaches 1000 crossed the tier boundary (values 999950-999999): promote to the M tier instead of rendering "1000k". The M branch above needs no such promotion -- M is the top tier. */}} {{- if ge $whole 1000.0 -}} {{- $result = "1M" -}} {{- else if eq $frac 0.0 -}} {{- $result = printf "%.0fk" $whole -}} {{- else -}} {{- $result = printf "%.0f.%.0fk" $whole $frac -}} {{- end -}} {{- end -}} {{- return $result -}}
