{{/* Performs a single HTTP attempt against a remote endpoint (arXiv Atom API, Semantic Scholar, or Hugging Face Papers) and normalizes the outcome into a stable result dict. This partial is the per-attempt unit driven by the outer retry loops in `arxiv-paper/fetch.html`. Why this partial exists: - Encapsulates the "build options + call resources.GetRemote inside try + unmarshal by explicit format + classify the failure" sequence so the three endpoints share one implementation. - Handles the arXiv media-type quirk: arXiv returns Content-Type application/atom+xml, which Hugo will not decode via a resource-typed transform.Unmarshal. We therefore unmarshal the raw `.Content` string with an EXPLICIT `format` option ("xml" for arXiv, "json" for the JSON APIs), which bypasses media-type inference entirely. - Note: GetRemote itself still fails (`.Err`) when the consuming site has not allow-listed application/atom+xml in security.http.mediaTypes. That error is caught here and classified as `media-type-config` so the caller can emit an actionable warning and degrade -- the build is never broken. Inputs (context dict): - .url (string) Full URL to fetch. - .cacheKey (string) Explicit per-attempt Hugo cache key so a response cached as an error on a prior attempt does not poison later attempts within the same build. - .format (string) "xml" or "json" -- the explicit decode format. - .headers (map, optional) Request headers (e.g., Authorization). - .timeout (string, optional) Per-request timeout (Go duration). Default "30s". - .responseHeaders (slice, optional) Response header names to expose via .Data. Output (returned dict): - .ok (bool) true when the response was 2xx and decoded cleanly. - .value (any) Decoded map/slice when .ok; nil otherwise. - .raw (string) Raw response body when a response was received. - .statusCode (int) HTTP status. 200 success; 0 network / media-type; 404 for the nil branch of resources.GetRemote. - .headers (map) Selected response headers when requested. - .errorClass (string) Empty when .ok; otherwise from classify-error.html. - .errorMessage (string) Short diagnostic when not .ok; empty on success. - .waitHintSeconds (int) Rate-limit wait hint; 0 elsewhere. Side effects: none. Diagnostic logging is the caller's responsibility. */}} {{- $url := .url -}} {{- $cacheKey := .cacheKey -}} {{- $format := .format | default "" -}} {{- $headers := .headers | default (dict) -}} {{- $timeout := .timeout | default "30s" -}} {{- $responseHeaders := .responseHeaders | default (slice) -}} {{- $opts := dict "method" "GET" "key" $cacheKey "timeout" $timeout -}} {{- with $headers -}} {{- $opts = merge $opts (dict "headers" .) -}} {{- end -}} {{- with $responseHeaders -}} {{- $opts = merge $opts (dict "responseHeaders" .) -}} {{- end -}} {{/* Default result: a failed attempt with no observable response. */}} {{- $result := dict "ok" false "value" nil "raw" "" "statusCode" 0 "headers" (dict) "errorClass" "network" "errorMessage" "no response (network or timeout)" "waitHintSeconds" 0 -}} {{- with try (resources.GetRemote $url $opts) -}} {{- with .Err -}} {{/* Error path. Media-type-resolution failures carry no .Cause.Data (no HTTP response object was built), so statusCode stays 0 and the classifier keys off the error string. HTTP errors expose status/headers/body via .Cause.Data. */}} {{- $errString := printf "%s" . -}} {{- $sc := 0 -}} {{- $hdrs := dict -}} {{- $body := "" -}} {{- with .Cause -}} {{- with .Data -}} {{- with .StatusCode }}{{ $sc = . }}{{ end -}} {{- with .Headers }}{{ $hdrs = . }}{{ end -}} {{- with .Body }}{{ $body = . }}{{ end -}} {{- end -}} {{- end -}} {{- $cls := partial "arxiv-paper/classify-error.html" (dict "statusCode" $sc "headers" $hdrs "body" $body "errString" $errString ) -}} {{- $result = dict "ok" false "value" nil "raw" $body "statusCode" $sc "headers" $hdrs "errorClass" $cls.errorClass "errorMessage" $cls.errorMessage "waitHintSeconds" $cls.waitHintSeconds -}} {{- else with .Value -}} {{/* Success path. Decode the raw content with an explicit format so arXiv's application/atom+xml is parsed as XML without media-type inference. A 2xx response that fails to decode is a hard parse error, not a transient. */}} {{- $raw := .Content -}} {{- $sucHdrs := dict -}} {{- with .Data -}} {{- with .Headers }}{{ $sucHdrs = . }}{{ end -}} {{- end -}} {{- $parsed := "" -}} {{- if $format -}} {{- $parsed = try (transform.Unmarshal (dict "format" $format) $raw) -}} {{- else -}} {{- $parsed = try (transform.Unmarshal $raw) -}} {{- end -}} {{- with $parsed.Err -}} {{- $result = dict "ok" false "value" nil "raw" $raw "statusCode" 200 "headers" $sucHdrs "errorClass" "parse" "errorMessage" (printf "unmarshal (%s) failed: %s" $format .) "waitHintSeconds" 0 -}} {{- else -}} {{- $result = dict "ok" true "value" $parsed.Value "raw" $raw "statusCode" 200 "headers" $sucHdrs "errorClass" "" "errorMessage" "" "waitHintSeconds" 0 -}} {{- end -}} {{- else -}} {{/* Nil branch: HTTP 404 (resources.GetRemote returns nil). */}} {{- $result = dict "ok" false "value" nil "raw" "" "statusCode" 404 "headers" (dict) "errorClass" "not-found" "errorMessage" "HTTP 404" "waitHintSeconds" 0 -}} {{- end -}} {{- end -}} {{- return $result -}}
