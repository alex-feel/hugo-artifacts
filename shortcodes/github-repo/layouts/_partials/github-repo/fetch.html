{{/* Fetches and normalizes GitHub repository data via the REST API with retry-and-classify resilience in front of the existing graceful degradation. When all attempts fail, the partial returns a fallback dict with `apiOk: false` and the variant partials degrade to the inline chip. The inline variant performs no fetch at all: it renders only URL-derived fields, so the partial returns the fallback dict directly and spends no API request budget. Why retries (and only this much): - resources.GetRemote in Hugo v0.160.1 has NO template-passed retry knobs (no retryWaitMin / retryWaitMax / retryMax). Hugo's INTERNAL retry covers only HTTP 408/429/500/502/503/504 -- it does NOT retry 403, the response GitHub uses for primary rate-limit. Hugo also does not honor Retry-After or X-RateLimit-Reset. - Hugo templates have no sleep primitive, so true backoff between outer attempts is structurally impossible. The closest approximation is to drive multiple attempts with fresh cache keys (each attempt bypasses Hugo's httpcache.Transport error caching) and to bound the whole thing by an outer wall-clock budget so a misbehaving endpoint cannot stall the build indefinitely. - On retry exhaustion the widget degrades to the inline chip exactly as it did before (variant partials' `else` branch -> inline.html). This behavior is preserved as the final fallback by user requirement. Retry constants (intentionally NOT shortcode parameters and NOT site params -- the user requirement explicitly forbids new override surface): - $attempts 5 Outer attempts per fetched endpoint. - $perAttemptTimeout 30s Per-request timeout for resources.GetRemote. - $overallBudgetSec 120 Wall-clock cap per fetched endpoint. An attempt only starts while the remaining budget still fits a full per-attempt timeout, so the cap is a hard ceiling -- never overshot by an attempt started near the boundary. - $waitHintCapSec 30 Display cap for waitHintSeconds in warnings; the full numeric hint is logged unconditionally so operators can act on it. Early-break classes -- subsequent attempts are provably useless within a single build, so the loop short-circuits when the prior attempt's errorClass is one of: - "primary-rate-limit" Bucket is exhausted; reset is wall-clock-bound. - "auth" Token/permissions issue; another attempt cannot fix. - "not-found" Resource genuinely missing. - "secondary-rate-limit" with waitHintSeconds > remaining budget. Host-down circuit breaker: a loop that exhausts with NO attempt receiving any HTTP status code (every attempt ended errorClass "network" with statusCode 0 -- a class the loop never early-breaks on, so its attempts were genuinely spent) marks api.github.com unreachable for the rest of the build via hugo.Store. Every later fetch loop in the build checks the sentinel first and degrades immediately with a warn, so a full API outage costs roughly one wall-clock budget per build instead of one per call site (concurrently rendered pages that started before the sentinel landed can each still pay an overlapping budget) -- sequential budget stacking within one page render is what pushes a page past Hugo's render `timeout` (see the README's "Interplay with Hugo's render timeout"). Any surfaced status code on any attempt suppresses the trip; note that Hugo retries the retryable statuses (408/429/500/502/503/504) internally within each attempt's request window, so a host answering ONLY with those for a whole window surfaces status-less and trips the breaker like a dark host -- intended, since it never yields a usable response within the budget. Context expectations: - .url (string) Full GitHub repository URL. - .variant (string) Display variant (inline, card, stats, lang, hero). - .name (string) Display name override (empty for default). - .description (string) Description override (empty for default). - .position (text.Position) Shortcode call position (used in warnings). Returns: a normalized dict with repository metadata, formatted values, and variant-specific data (`languages` for `lang`, `sparkline` for `hero`). The dict's `apiOk` flag is the contract the variant partials branch on. */}} {{- $url := .url -}} {{- $variant := .variant -}} {{- $nameOverride := .name -}} {{- $descOverride := .description -}} {{- $position := .position -}} {{/* Parse owner/repo from URL. */}} {{- $path := replaceRE `^https?://github\.com/([^/]+/[^/]+)/?.*$` "$1" $url -}} {{- $parts := split $path "/" -}} {{- $owner := index $parts 0 -}} {{- $repo := index $parts 1 -}} {{/* Build fallback dict for API failure. The variant partials branch on `apiOk` and degrade to inline.html when it is false. */}} {{- $fallback := dict "owner" $owner "repo" $repo "fullName" (printf "%s/%s" $owner $repo) "name" (or $nameOverride $repo) "description" (or $descOverride "") "language" "" "langColor" "" "stars" 0 "starsCount" 0 "starsFormatted" "0" "forks" 0 "forksCount" 0 "forksFormatted" "0" "watchers" 0 "license" "" "topics" (slice) "updated" "" "url" $url "apiOk" false "languages" (slice) "sparkline" (slice) "extraClass" "" -}} {{/* Resolve the token once. Hugo's default security policy only exposes env vars matching ^HUGO_ or ^CI$, so the prefix is mandatory. */}} {{- $token := os.Getenv "HUGO_GITHUB_TOKEN" -}} {{/* Retry constants. Aggressive defaults per user-approved plan. */}} {{- $attempts := 5 -}} {{- $perAttemptTimeoutSec := 30 -}} {{- $perAttemptTimeout := printf "%ds" $perAttemptTimeoutSec -}} {{- $overallBudgetSec := 120 -}} {{- $waitHintCapSec := 30 -}} {{/* Early-break classes shared across the three endpoint loops. */}} {{- $earlyBreakClasses := slice "primary-rate-limit" "auth" "not-found" -}} {{/* Host-down circuit-breaker sentinel, shared across every call site in the build. Each endpoint loop snapshots it before starting and trips it after exhausting its attempts against a host that never responded. */}} {{- $hostDownKey := "github-repo:host-down:api.github.com" -}} {{/* API base URL. */}} {{- $apiBase := printf "https://api.github.com/repos/%s/%s" $owner $repo -}} {{/* Normalized data dict; starts as the fallback and is replaced on a successful fetch. Deduplication uses hugo.Store (build-scoped scratch) -- the first call observes nil for the sentinel key and warns; subsequent calls observe the sentinel and stay silent. */}} {{- if not $token -}} {{- $warnedKey := "github-repo:warned-no-token" -}} {{- if not (hugo.Store.Get $warnedKey) -}} {{- hugo.Store.Set $warnedKey true -}} {{- warnf "[github-repo] HUGO_GITHUB_TOKEN is not set. Unauthenticated GitHub API requests are limited to 60 per hour, which can cause shortcode renders to degrade to the inline chip during a build. Set HUGO_GITHUB_TOKEN to a personal access token (no scopes required for public repositories) to raise the limit to 5,000 per hour. See %s" $position -}} {{- end -}} {{- end -}} {{/* ---------- Endpoint 1 of up to 3: base repo ---------- */}} {{- $hostDown := hugo.Store.Get $hostDownKey -}} {{- $startSec := int now.Unix -}} {{- $repoResult := dict "ok" false "value" nil "statusCode" 0 "headers" (dict) "errorClass" "network" "errorMessage" "" "waitHintSeconds" 0 -}} {{- $repoAttemptsUsed := 0 -}} {{- $repoSawResponse := false -}} {{- range $attempt := seq 1 $attempts -}} {{- if $repoResult.ok -}} {{/* Already succeeded; skip remaining attempts. */}} {{- else if $hostDown -}} {{/* Circuit breaker open: the host never responded to an earlier call site this build, so degrade immediately instead of burning another wall-clock budget. */}} {{- else -}} {{- $elapsed := sub (int now.Unix) $startSec -}} {{- $remaining := sub $overallBudgetSec $elapsed -}} {{- if lt $remaining $perAttemptTimeoutSec -}} {{/* The remaining budget cannot fit a full attempt; stop so the budget stays a hard ceiling. */}} {{- else if and (gt $attempt 1) (in $earlyBreakClasses $repoResult.errorClass) -}} {{/* Provably-useless class; further attempts within this build cannot succeed. */}} {{- else if and (gt $attempt 1) (eq $repoResult.errorClass "secondary-rate-limit") (gt $repoResult.waitHintSeconds $remaining) -}} {{/* Secondary rate-limit reset window exceeds remaining budget. */}} {{- else -}} {{- $cacheKey := printf "github-repo:%s/%s:repo:attempt%d" $owner $repo $attempt -}} {{- $repoResult = partial "github-repo/fetch-once.html" (dict "url" $apiBase "cacheKey" $cacheKey "token" $token "timeout" $perAttemptTimeout ) -}} {{- $repoAttemptsUsed = $attempt -}} {{- if ne $repoResult.statusCode 0 -}} {{- $repoSawResponse = true -}} {{- end -}} {{- end -}} {{- end -}} {{- end -}} {{/* Trip the circuit breaker only when NO attempt received any HTTP status code -- a host that answered even once (with any status) is reachable, and the problem is then the endpoint or the request, not the host. */}} {{- if and (not $hostDown) (not $repoResult.ok) (eq $repoResult.errorClass "network") (eq $repoResult.statusCode 0) (not $repoSawResponse) -}} {{- hugo.Store.Set $hostDownKey true -}} {{- end -}} {{/* On success, populate the normalized data dict from the JSON response; on breaker skip, warn that this call site degraded without spending budget; on failure, keep the fallback and emit a single structured warning so operators can act. */}} {{- if $repoResult.ok -}} {{- $json := $repoResult.value -}} {{/* Compact number formatting. */}} {{- $starsRaw := or $json.stargazers_count 0 -}} {{- $forksRaw := or $json.forks_count 0 -}} {{- $starsF := partial "github-repo/compact-number.html" $starsRaw -}} {{- $forksF := partial "github-repo/compact-number.html" $forksRaw -}} {{/* Guarded integer counts for the i18n plural-form selection: the raw values are remote-derived, so the cast is try-wrapped and a bad value degrades to 0 instead of failing the build. */}} {{- $starsN := 0 -}} {{- $t := try (int $starsRaw) -}} {{- if not $t.Err -}}{{- with $t.Value -}}{{- $starsN = . -}}{{- end -}}{{- end -}} {{- $forksN := 0 -}} {{- $t = try (int $forksRaw) -}} {{- if not $t.Err -}}{{- with $t.Value -}}{{- $forksN = . -}}{{- end -}}{{- end -}} {{/* Relative time from pushed_at. */}} {{- $updatedStr := "" -}} {{- with $json.pushed_at -}} {{- $updatedStr = partial "github-repo/relative-time.html" . -}} {{- end -}} {{/* Language color lookup. */}} {{- $lang := or $json.language "" -}} {{- $langColor := "" -}} {{- with $lang -}} {{- with index hugo.Data.github_repo_lang_colors . -}} {{- $langColor = . -}} {{- end -}} {{- end -}} {{/* License extraction. */}} {{- $license := "" -}} {{- with $json.license -}} {{- with .spdx_id -}} {{- if ne . "NOASSERTION" -}} {{- $license = . -}} {{- end -}} {{- end -}} {{- end -}} {{- $data = dict "owner" $owner "repo" $repo "fullName" (printf "%s/%s" $owner $repo) "name" (or $nameOverride $json.name $repo) "description" (or $descOverride $json.description "") "language" $lang "langColor" $langColor "stars" $starsRaw "starsCount" $starsN "starsFormatted" $starsF "forks" $forksRaw "forksCount" $forksN "forksFormatted" $forksF "watchers" (or $json.subscribers_count 0) "license" $license "topics" (or $json.topics (slice)) "updated" $updatedStr "url" $url "apiOk" true "languages" (slice) "sparkline" (slice) "extraClass" "" -}} {{- else if $hostDown -}} {{- warnf "[github-repo] Skipping the GitHub API fetch for %s/%s: api.github.com was marked unreachable earlier in this build. Rendering the inline chip. See %s" $owner $repo $position -}} {{- else -}} {{- $hint := $repoResult.waitHintSeconds -}} {{- $guidanceHint := "" -}} {{- if gt $hint 0 -}} {{- if gt $hint $waitHintCapSec -}} {{- $guidanceHint = printf "wait %d seconds and rebuild (display capped at %ds)" $hint $waitHintCapSec -}} {{- else -}} {{- $guidanceHint = printf "wait %d seconds and rebuild" $hint -}} {{- end -}} {{- else -}} {{- $guidanceHint = "see warning context for guidance" -}} {{- end -}} {{- warnf "[github-repo] Failed to fetch %s/%s after %d attempt(s) (errorClass=%s, statusCode=%d, message=%q). Hint: %s. See %s" $owner $repo $repoAttemptsUsed $repoResult.errorClass $repoResult.statusCode $repoResult.errorMessage $guidanceHint $position -}} {{- end -}} {{/* Closes the non-inline guard around the token preflight and endpoint 1. */}} {{- end -}} {{/* ---------- Endpoint 2: language composition (variant=lang only) ---------- */}} {{- if and (eq $variant "lang") $data.apiOk -}} {{- $langHostDown := hugo.Store.Get $hostDownKey -}} {{- $langUrl := printf "%s/languages" $apiBase -}} {{- $langStartSec := int now.Unix -}} {{- $langResult := dict "ok" false "value" nil "statusCode" 0 "headers" (dict) "errorClass" "network" "errorMessage" "" "waitHintSeconds" 0 -}} {{- $langAttemptsUsed := 0 -}} {{- $langSawResponse := false -}} {{- range $attempt := seq 1 $attempts -}} {{- if $langResult.ok -}} {{/* Already succeeded; skip remaining attempts. */}} {{- else if $langHostDown -}} {{/* Circuit breaker open; degrade immediately. */}} {{- else -}} {{- $elapsed := sub (int now.Unix) $langStartSec -}} {{- $remaining := sub $overallBudgetSec $elapsed -}} {{- if lt $remaining $perAttemptTimeoutSec -}} {{/* The remaining budget cannot fit a full attempt. */}} {{- else if and (gt $attempt 1) (in $earlyBreakClasses $langResult.errorClass) -}} {{/* Provably-useless class. */}} {{- else if and (gt $attempt 1) (eq $langResult.errorClass "secondary-rate-limit") (gt $langResult.waitHintSeconds $remaining) -}} {{/* Reset window exceeds remaining budget. */}} {{- else -}} {{- $cacheKey := printf "github-repo:%s/%s:languages:attempt%d" $owner $repo $attempt -}} {{- $langResult = partial "github-repo/fetch-once.html" (dict "url" $langUrl "cacheKey" $cacheKey "token" $token "timeout" $perAttemptTimeout ) -}} {{- $langAttemptsUsed = $attempt -}} {{- if ne $langResult.statusCode 0 -}} {{- $langSawResponse = true -}} {{- end -}} {{- end -}} {{- end -}} {{- end -}} {{- if and (not $langHostDown) (not $langResult.ok) (eq $langResult.errorClass "network") (eq $langResult.statusCode 0) (not $langSawResponse) -}} {{- hugo.Store.Set $hostDownKey true -}} {{- end -}} {{- if $langResult.ok -}} {{- $langMap := $langResult.value -}} {{- $total := 0.0 -}} {{- range $lang, $bytes := $langMap -}} {{- $total = math.Add $total (float $bytes) -}} {{- end -}} {{- $langSlice := slice -}} {{- if gt $total 0 -}} {{- range $lang, $bytes := $langMap -}} {{- $pct := math.Round (math.Mul (math.Div (float $bytes) $total) 100) -}} {{- $color := "#858585" -}} {{- with index hugo.Data.github_repo_lang_colors $lang -}} {{- $color = . -}} {{- end -}} {{- $langSlice = $langSlice | append (dict "name" $lang "pct" $pct "color" $color) -}} {{- end -}} {{- $langSlice = sort $langSlice "pct" "desc" -}} {{- end -}} {{- $data = merge $data (dict "languages" $langSlice) -}} {{- else if $langHostDown -}} {{- warnf "[github-repo] Skipping the languages fetch for %s/%s: api.github.com was marked unreachable earlier in this build. Rendering without the language composition. See %s" $owner $repo $position -}} {{- else -}} {{- $hint := $langResult.waitHintSeconds -}} {{- $guidanceHint := "" -}} {{- if gt $hint 0 -}} {{- if gt $hint $waitHintCapSec -}} {{- $guidanceHint = printf "wait %d seconds and rebuild (display capped at %ds)" $hint $waitHintCapSec -}} {{- else -}} {{- $guidanceHint = printf "wait %d seconds and rebuild" $hint -}} {{- end -}} {{- else -}} {{- $guidanceHint = "see warning context for guidance" -}} {{- end -}} {{- warnf "[github-repo] Failed to fetch %s/%s languages after %d attempt(s) (errorClass=%s, statusCode=%d, message=%q). Hint: %s. See %s" $owner $repo $langAttemptsUsed $langResult.errorClass $langResult.statusCode $langResult.errorMessage $guidanceHint $position -}} {{- end -}} {{- end -}} {{/* ---------- Endpoint 3: participation sparkline (variant=hero only) ---------- */}} {{- if and (eq $variant "hero") $data.apiOk -}} {{- $statsHostDown := hugo.Store.Get $hostDownKey -}} {{- $statsUrl := printf "%s/stats/participation" $apiBase -}} {{- $statsStartSec := int now.Unix -}} {{- $statsResult := dict "ok" false "value" nil "statusCode" 0 "headers" (dict) "errorClass" "network" "errorMessage" "" "waitHintSeconds" 0 -}} {{- $statsAttemptsUsed := 0 -}} {{- $statsSawResponse := false -}} {{- range $attempt := seq 1 $attempts -}} {{- if $statsResult.ok -}} {{/* Already succeeded; Rendering without the sparkline. See %s" $owner $repo $position -}} {{- else -}} {{- $hint := $statsResult.waitHintSeconds -}} {{- $guidanceHint := "" -}} {{- if gt $hint 0 -}} {{- if gt $hint $waitHintCapSec -}} {{- $guidanceHint = printf "wait %d seconds and rebuild (display capped at %ds)" $hint $waitHintCapSec -}} {{- else -}} {{- $guidanceHint = printf "wait %d seconds and rebuild" $hint -}} {{- end -}} {{- else -}} {{- $guidanceHint = "see warning context for guidance" -}} {{- end -}} {{- warnf "[github-repo] Failed to fetch %s/%s participation after %d attempt(s) (errorClass=%s, statusCode=%d, message=%q). Hint: %s. See %s" $owner $repo $statsAttemptsUsed $statsResult.errorClass $statsResult.statusCode $statsResult.errorMessage $guidanceHint $position -}} {{- end -}} {{- end -}} {{- return $data -}}
