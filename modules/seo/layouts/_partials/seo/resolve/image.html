{{/* Normalizes ONE image candidate to a stable dict {url, width, height, type, alt, natural}. Pure value chooser: no HTML, no side effects. Context (dict): - .candidate (any) Either a Hugo resource (page-bundle or global) OR a string (bundle-relative path, site-root path, or absolute/remote URL). - .page (page) The current page, for the alt fallback chain and for resolving a string candidate as a bundle or global resource. - .variant (string) Optional. "og" (default): a local raster gets the content-aware 1.91:1 crop that og:image wants. "natural": no crop; the source image's own dimensions are kept -- for JSON-LD entity images (Organization logo, ProfilePage mainEntity image, author Person image), where a 1.91:1 crop destroys logos and portraits. Normalization rules: - Local raster resource (jpg/png/webp/gif/...), variant "og": .Fill "1200x630 Smart" then .Permalink (absolute), .Width (1200), .Height (630), .MediaType.Type. This yields real og:image:width/height/type and a content-aware 1.91:1 crop. - Local raster resource, variant "natural": the source resource's own .Permalink/.Width/.Height, no processing. - SVG resource: passed through WITHOUT processing (Hugo cannot raster-process SVG); url + type only, width/height omitted. - String candidate (unless absolute or protocol-relative): resolved as a page-bundle resource first ($page.Resources.GetMatch, so bundle-relative values like "tile.png" work), else as a global resource (resources.Get). A string that resolves to neither is treated as a bare URL/path: absolutized, no processing, NO fabricated dimensions; type inferred from the extension only when unambiguous (.png/.jpg/.jpeg/.webp/.gif/.svg). natural sub-dict: every resolved image carries `natural` -- the un-cropped source values {url, width, height} (dimensions present only for local raster resources) -- so a consumer that needs the native aspect can read it regardless of the variant that shaped the top-level values. alt chain: seo.image_alt -> resource .Params.alt -> resource .Title (when set to something other than the filename) -> params.seo.image_alt -> resolved page title. So og:image:alt / twitter:image:alt are ALWAYS present when an image is. Returns: {url, width, height, type, alt, natural}, or an empty dict when the candidate is unresolvable. ONE terminal return. */}} {{- $candidate := .candidate -}} {{- $page := .page -}} {{- $variant := .variant | default "og" -}} {{- $result := dict -}} {{- $res := "" -}} {{/* Resolve the candidate to a resource when possible. */}} {{- if reflect.IsResource $candidate -}} {{- $res = $candidate -}} {{- else if $candidate -}} {{/* String candidate: a bundle-relative path resolves as a page resource (the more specific scope wins), a project path as a global resource; otherwise it is treated as a bare URL/path below. */}} {{- $s := printf "%v" $candidate -}} {{- if not (or (hasPrefix $s "http://") (hasPrefix $s "https://") (hasPrefix $s "//")) -}} {{- with $page.Resources.GetMatch $s -}} {{- $res = . -}} {{- end -}} {{- if not $res -}} {{- with resources.Get $s -}} {{- $res = . -}} {{- end -}} {{- end -}} {{- end -}} {{- end -}} {{/* alt chain (shared by the resource and string branches). */}} {{- $alt := "" -}} {{- with $page.Param "seo.image_alt" -}} {{- $alt = . -}} {{- end -}} {{- if $res -}} {{/* Resource branch: distinguish raster from SVG by sub-type. */}} {{- $isImage := eq $res.ResourceType "image" -}} {{- $isSVG := and $isImage (eq $res.MediaType.SubType "svg") -}} {{- if not $alt -}} {{- with $res.Params.alt -}} {{- $alt = . -}} {{- else -}} {{/* Use resource .Title only when it is not merely the filename. */}} {{- $rt := $res.Title -}} {{- if and $rt (ne $rt $res.Name) -}} {{- $alt = $rt -}} {{- end -}} {{- end -}} {{- end -}} {{- if and $isImage (not $isSVG) -}} {{/* Local raster: real dimensions. The og variant adds the content-aware 1.91:1 crop; the natural variant keeps the source as-is. */}} {{- $natural := dict "url" $res.Permalink "width" $res.Width "height" $res.Height -}} {{- $chosen := $natural -}} {{- if eq $variant "og" -}} {{- $filled := $res.Fill "1200x630 Smart" -}} {{- $chosen = dict "url" $filled.Permalink "width" $filled.Width "height" $filled.Height -}} {{- end -}} {{- $result = dict "url" $chosen.url "width" $chosen.width "height" $chosen.height "type" $res.MediaType.Type "alt" $alt "natural" $natural -}} {{- else -}} {{/* SVG (or a non-raster resource): url + type only, no dimensions. */}} {{- $result = dict "url" $res.Permalink "type" $res.MediaType.Type "alt" $alt "natural" (dict "url" $res.Permalink) -}} {{- end -}} {{- else if $candidate -}} {{/* String branch: bare URL or site-root path, absolutized, no dimensions. */}} {{- $s := printf "%v" $candidate -}} {{- $url := $s | absURL -}} {{- $type := "" -}} {{- $lower := lower $s -}} {{- if hasSuffix $lower ".png" -}} {{- $type = "image/png" -}} {{- else if or (hasSuffix $lower ".jpg") (hasSuffix $lower ".jpeg") -}} {{- $type = "image/jpeg" -}} {{- else if hasSuffix $lower ".webp" -}} {{- $type = "image/webp" -}} {{- else if hasSuffix $lower ".gif" -}} {{- $type = "image/gif" -}} {{- else if hasSuffix $lower ".svg" -}} {{- $type = "image/svg+xml" -}} {{- end -}} {{- $result = dict "url" $url "alt" $alt "natural" (dict "url" $url) -}} {{- with $type -}} {{- $result = merge $result (dict "type" .) -}} {{- end -}} {{- end -}} {{/* Final alt fallbacks apply only when an image actually resolved. */}} {{- if $result.url -}} {{- if not $result.alt -}} {{- $altFallback := "" -}} {{- with $page.Param "seo.image_alt" -}} {{- $altFallback = . -}} {{- else -}} {{- $t := partial "seo/resolve/title.html" $page -}} {{- $altFallback = $t.title -}} {{- end -}} {{- $result = merge $result (dict "alt" $altFallback) -}} {{- end -}} {{- end -}} {{- return $result -}}
