{{/* Builds the ordered, de-duplicated slice of normalized image dicts for the page. images[0] is the primary OG/Twitter image; the full slice feeds JSON-LD image[]/thumbnailUrl[]. Pure value chooser: no HTML, no side effects. Context: the current Page, passed as the dot. Candidate order (explicit declarations outrank filename-convention detection): 1. seo.images (list) or seo.image (single) or meta_image (alias) -- explicit. 2. Site-configurable extra keys: params.seo.image_params names top-level front-matter params (e.g. image_params = ["tile_image"]) whose values -- a single string or a list -- become candidates, so a site's own image vocabulary feeds the pipeline without overriding this resolver. 3. Native .Params.images ([]string), in order. 4. Page-bundle convention match, DETERMINISTIC priority: GetMatch "*feature*", then "*cover*", then "*thumbnail*" (feature beats cover beats thumbnail). 5. params.seo.default_image (alias params.metadata.image) as the site fallback -- applied ONLY when steps 1-4 RESOLVED no image, so a page whose declared candidates all fail to resolve still gets the default. Each candidate is normalized through resolve/image.html (which itself resolves bundle-relative paths, global resources, and bare URLs) and deduped by url. Returns: an ordered slice of {url,width,height,type,alt,natural} dicts (possibly empty). ONE terminal return. */}} {{- $page := . -}} {{- $candidates := slice -}} {{/* 1. Explicit SEO override: seo.images / seo.image / meta_image. */}} {{- with $page.Param "seo.images" -}} {{- $candidates = $candidates | append . -}} {{- else with $page.Param "seo.image" -}} {{- $candidates = $candidates | append . -}} {{- else with $page.Params.meta_image -}} {{- $candidates = $candidates | append . -}} {{- end -}} {{/* 2. Site-configurable extra front-matter keys (params.seo.image_params): each named top-level param contributes its value or values. */}} {{- with $page.Param "seo.image_params" -}} {{- range . -}} {{- $key := lower (printf "%v" .) -}} {{- with index $page.Params $key -}} {{- if reflect.IsSlice . -}} {{- range . -}} {{- $candidates = $candidates | append . -}} {{- end -}} {{- else -}} {{- $candidates = $candidates | append . -}} {{- end -}} {{- end -}} {{- end -}} {{- end -}} {{/* 3. Native .Params.images; resolve/image.html resolves each entry as a bundle resource, a global resource, or a path/URL. */}} {{- with $page.Params.images -}} {{- range . -}} {{- $candidates = $candidates | append . -}} {{- end -}} {{- end -}} {{/* 4. Deterministic page-bundle convention: feature -> cover -> thumbnail. */}} {{- with $page.Resources -}} {{- with $page.Resources.GetMatch "*feature*" -}} {{- $candidates = $candidates | append . -}} {{- end -}} {{- with $page.Resources.GetMatch "*cover*" -}} {{- $candidates = $candidates | append . -}} {{- end -}} {{- with $page.Resources.GetMatch "*thumbnail*" -}} {{- $candidates = $candidates | append . -}} {{- end -}} {{- end -}} {{/* Normalize each candidate and dedup by url, preserving order. */}} {{- $images := slice -}} {{- $seen := slice -}} {{- range $candidates -}} {{- $img := partial "seo/resolve/image.html" (dict "candidate" . "page" $page) -}} {{- with $img.url -}} {{- if not (in $seen .) -}} {{- $seen = $seen | append . -}} {{- $images = $images | append $img -}} {{- end -}} {{- end -}} {{- end -}} {{/* 5. Site-wide default image (alias params.metadata.image) -- a TRUE fallback: applied when nothing RESOLVED above, so pages with their own representative images never carry the site default in JSON-LD image[], while a page whose declared candidates all failed to resolve still gets the default. */}} {{- if not $images -}} {{- $default := "" -}} {{- with $page.Param "seo.default_image" -}} {{- $default = . -}} {{- else with $page.Param "metadata.image" -}} {{- $default = . -}} {{- end -}} {{- with $default -}} {{- $img := partial "seo/resolve/image.html" (dict "candidate" . "page" $page) -}} {{- with $img.url -}} {{- $images = $images | append $img -}} {{- end -}} {{- end -}} {{- end -}} {{- return $images -}}
